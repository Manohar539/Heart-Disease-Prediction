Project Overview

This project presents a cloud-ready machine learning application capable of predicting cardiovascular risk using three independent healthcare datasets. Each dataset was preprocessed, analysed, trained, evaluated and deployed using its best-performing machine learning model.

Datasets Used

Dataset Prediction Goal Records Features
Framingham Heart Study 10-Year Coronary Heart Disease Risk 4,240 15
Heart Disease Dataset Heart Disease Classification 920 12
Heart Failure Clinical Records Heart Failure Outcome Prediction 5,000 12

Academic Disclaimer

This application has been developed exclusively for academic research and educational demonstration purposes as part of a Master's programme. The predictions generated by the models are intended to illustrate machine learning techniques and must not be interpreted as professional medical advice or clinical diagnosis.
