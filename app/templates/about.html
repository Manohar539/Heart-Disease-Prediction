❤️ About the Heart Disease Prediction Project

National College of Ireland
Cloud Machine Learning Project

Project Objective

The objective of this project is to develop a cloud-ready machine learning application that predicts the risk of heart disease using demographic, behavioural, and clinical patient information.

Datasets Analysed

Three publicly available cardiovascular datasets were collected and compared:

  • Framingham Heart Study Dataset — selected as the primary training dataset.
  • Heart Disease Dataset — used for comparative feature and target analysis.
  • Heart Failure Clinical Records Dataset — used for comparative analysis of cardiovascular outcomes.

The datasets were compared based on record count, feature availability, missing values, clinical variables, and prediction objectives. The Framingham dataset was selected because it contains a broad set of cardiovascular risk factors and aligns most closely with the goal of predicting ten-year coronary heart disease risk.

Machine Learning Models

  • Logistic Regression
  • Random Forest
  • Random Forest with SMOTE
  • XGBoost

The models were evaluated using accuracy, precision, recall, F1-score, confusion matrices, and cross-validation.

Random Forest was selected for deployment because it achieved the strongest recall and F1-score for identifying heart disease cases among the evaluated baseline models.
